# Subaward Aggregator

A .NET console app for extracting and aggregating subaward data from Excel spreadsheets.

## Usage

```bash
dotnet run --project src/UiucTakeHome.Console -- --folder ./samples
```

Add `--verbose` to show amounts under each file listing.

## Testing

```bash
dotnet test
```
